Noteshare is a website that allows users to share notes with peers electronically. You can upload your notes and assign a subject and various tags you want to be associated with your file. Others can then search for notes using various parameters, such as file name, date uploaded, user uploaded, rating, file extension, subject, and tags. For more information, click on the information tab located on top.
